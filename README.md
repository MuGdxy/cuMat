# cuMat: Linear algebra library in CUDA

Features:
 - Expression templates for kernel fusion
 - Header-only library
 - Thread-save, supports multiple parallel threads and CUDA devices
 - API very similar to Eigen
 - Eigen-Interop
 - BLAS 1-3
